Attributes

The HTML attributes are used to provide more information to an HTML element
  • HTML attributes apperd at small tags. It will never be on end tags.
  • All HTML element can have attributes.
  • HTML element can have multipal attributes.
  • HTML attributes usually come in name/value pairs like:"value"
    • An attributes name
    • An equl=sign
    • A value surrounded by quotation mark"value"
You can also use single quotation marks depending on the situation esp. when the value contains double quote.
