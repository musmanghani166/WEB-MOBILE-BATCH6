Empty Elements
Empty Elements are elementthat do not have an element content and an end tag.

The <br> tag define a line break, and is an empty element without closing tag.

Code:
<p>This a <br />paragraph with a line break<p>
Output:

This is a
paragraph with a line break

A list of commonly used Empty Element

  1. <meta />
  2. <link />
  3. <img />
  4. <br />
  5. <hr />
  6. <input />
The best prectice in HTMLEmpty Element is to alweys put a forword slash / sign before the greter than >sign
In this way they are closed at their start tag are also called self closing tag.
